# calcul-gains-petanque

Version actuelle : v0.2.3

Cette application web simple aide à préparer la répartition des gains d’un concours de pétanque sans backend.

## Utilisation

1. Ouvrez le fichier index.html dans votre navigateur.
2. Saisissez le nombre d’équipes, l’inscription par équipe et la dotation de l’organisateur.
3. Choisissez un style de répartition : équilibré, finale plus forte, répartition plus large ou personnalisé.
4. Choisissez le type de concours : tête-à-tête, doublette ou triplette.
5. Cliquez sur « Calculer automatiquement ».
6. Consultez l’enveloppe totale, le gain total du vainqueur, le tableau des tours et la liste « À annoncer ».
7. Consultez la section « Partage possible de la cagnotte restante » pour estimer un partage anticipé entre les équipes encore en course.

En mode personnalisé, les gains non divisibles par le nombre de joueurs de l’équipe restent autorisés mais sont signalés par un avertissement orange.

## Exemple

Pour 50 équipes, une inscription de 15 € et une dotation de 300 €, l’enveloppe totale est de 1050 €.
