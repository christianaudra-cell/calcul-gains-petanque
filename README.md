# calcul-gains-petanque

Cette application web simple aide à préparer la répartition des gains d’un concours de pétanque sans backend.

## Utilisation

1. Ouvrez le fichier index.html dans votre navigateur.
2. Saisissez le nombre d’équipes, l’inscription par équipe et la dotation de l’organisateur.
3. Choisissez un style de répartition : équilibré, finale plus forte, répartition plus large ou personnalisé.
4. Cliquez sur « Calculer automatiquement ».
5. Consultez l’enveloppe totale, le gain total du vainqueur, le tableau des tours et la liste « À annoncer ».

## Exemple

Pour 50 équipes, une inscription de 15 € et une dotation de 300 €, l’enveloppe totale est de 1050 €.
